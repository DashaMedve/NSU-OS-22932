Этап 1
1. Перейти по ссылке https://github.com/vkinsu/NSU-OS-22932
2. В правом верхнем углу нажать на Fork -> Create new fork...
3. Теперь в профиле есть копия этого репозитория

Этап 2
1. Запустить GitBash и перейти в папку, подключенную к Github 
   (с которой брали SSH-ключ)
2. Ввести команду 
   git clone https://github.com/vkinsu/NSU-OS-22932
3. Перейти в склонированную папку в GitBash (cd NSU-OS-22932)
4. git init
5. Создать любой файл через проводник, можно пустой
6. Ввести "git add ."
7. git commit -m "тема сообщение"
8. git push

Этап 3
1. Зайти в своем профиле GitHub в скопированный репозиторий 
2. В левом верхнем углу Pull requests -> New pull request
   -> Create pull request
3. После этого остается дождаться подтверждения от владельца репозитория
